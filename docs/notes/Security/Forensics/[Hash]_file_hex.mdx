# [Hash] Exam file hex

## Online viewer
https://hexed.it/

## In linux with `xxd`

### Basic usage
- Output `fa0c` ect
```bash
xxd hello.txt
```

### Binary Display
- Output `00101101` ect
```bash
xxd -b datafile.txt
```

### Hex Upper Case
- Output `FA0C` ect
```bash
xxd -u datafile.txt
```
